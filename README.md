# AI Customer Support Email Bot using n8n + Gemini AI

An AI-powered customer support automation workflow built using n8n and Google Gemini AI.

## Features

- Automatically monitors Gmail inbox
- Uses Gemini AI to generate professional customer support replies
- Replies to customer emails automatically
- Human-like email responses
- Gmail integration with AI automation
- Smart support workflow system

---

## Workflow

Gmail Trigger
→ AI Agent
→ Gemini AI
→ Gmail Reply

---

## Tech Stack

- n8n
- Google Gemini AI
- Gmail API
- AI Agent Workflow

---

## Use Cases

- Customer support automation
- AI email assistant
- Business email handling
- Startup support automation
- FAQ auto response system
